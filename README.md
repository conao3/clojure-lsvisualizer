# lsvisualizer

A Clojure tool for visualizing Language Server Protocol (LSP) interactions and data.

## Requirements

- Clojure 1.12.0 or later

## Installation

Clone the repository:

```bash
git clone https://github.com/conao3/clojure-lsvisualizer.git
cd clojure-lsvisualizer
```

## Usage

Run the application:

```bash
clojure -M -m lsvisualizer.core
```

## Development

Start a REPL with development dependencies:

```bash
clojure -M:dev
```

Run tests using Kaocha:

```bash
clojure -M:test
```
